# Tic-Tac-Toe
 Tic Tac Toe game made with C# and Windows Forms. The game features a 2 player mode, allowing players to play against each other.

## Run

open the excutable at

```bash
  Tic-Tac-Toe/bin/Debug/Tic Tac Toe 2.0.exe
```
